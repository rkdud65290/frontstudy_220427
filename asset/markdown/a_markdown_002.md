# 절대주소, 상대주소

## 절대주소는 
**변하지 않는 주소** 로서
`c:` , `d:` 형태 또는 `http://naver.com`,
`/`는 디렉토리(폴더)를 의미하는 해석하며,
`/desktop/file.jpg` 경로상의 앞에 붙이면,
이동할 수 있는 가장 최상의 위치를 의미

## 상대주소는
**상대적인 주소** 로서
작성하고 있는 문서에서 `./`,
현재문서의 상위디렉토리 `../`
의 형태로 앞에 `.`의 표기가 1개 또는 2개로 시작되는 경로 주소표시
